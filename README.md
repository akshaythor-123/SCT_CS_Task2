# 🔐 Image Encryption Tool

## 📌 Project Overview

This project is a simple Image Encryption Tool developed in Python as part of the SkillCraft Technology Cyber Security Internship (Task 02).

The application encrypts and decrypts images using pixel manipulation techniques. A secret key is applied to each pixel's RGB values during encryption, and the same key is used to restore the original image.

---

## ✨ Features

- Encrypt images using a secret key
- Decrypt encrypted images
- Supports JPG, JPEG, PNG and BMP images
- User-friendly file picker
- Input validation
- Simple menu-driven interface
- Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python 3
- Pillow (PIL)
- Tkinter

## 📖 How It Works

1. Select an image.
2. Enter a secret key.
3. The program encrypts the image by modifying RGB pixel values.
4. To decrypt, select the encrypted image and enter the same secret key.
5. The original image is restored.
